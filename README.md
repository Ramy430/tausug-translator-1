## üèùÔ∏è About The Project

**Tausug Translator** is a digital preservation tool for the **Tausug language** (Bahasa S≈´g), an Austronesian language spoken by the Tausug people in the Sulu Archipelago of the Philippines and parts of Malaysia.

This web application provides:
- üî§ **Bidirectional translation** between Tausug and English
- üìö **Community-driven dictionary** that grows with contributions
- üîä **Text-to-speech** for pronunciation learning
- üíæ **Personal dictionary** storage in your browser
- ü§ù **GitHub integration** for submitting new words

> *"A language is not just words. It's a culture, a tradition, a unification of a community."*
